# 沖縄語辞典　電子データ
このレポジトリは、国立国語研究所の沖縄語辞典を電子アプリ化するために、同研究所の Web サイト上にある、沖縄語辞典の Excel 化されたデータを、アプリ作成で使いやすいように修正・変換するためのレポジトリです。

「国立国語研究所/沖縄語辞典 データ集」：
https://mmsrv.ninjal.ac.jp/okinawago/
